A Python based utility app for receiving, booking and managing calendar schedules. 

## Technology Stack
 
 - Python
 - MySQL
 - Django
 - JavaScript
 - Full Calendar
 - Bootstrap

## Set up Instructions
  - Clone repo
  - Create Database and update credentials 
  - Run migrations
  - Run test server with `python manage.py runserver`
